Interview Preparation App
Overview

Interview Preparation App is a simple web application built using HTML, CSS, and JavaScript. It helps users practice interview questions from different technology categories and improve their technical knowledge.

Features
Multiple categories such as HTML, CSS, JavaScript, Python, Java, and Django
Random question generation
Show answer functionality
Simple and user-friendly interface
Responsive design
Technologies Used
HTML5
CSS3
JavaScript

How to Run
Download or clone the repository.
Open the project folder.
Open index.html in a web browser.

Or clone using Git:

git clone https://github.com/G-Nishanthi21/Interview-Preparation.git
